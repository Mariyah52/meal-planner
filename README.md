# Smart Diet & Macro Planner Dashboard 🎯

A premium, dark-themed responsive web portal designed specifically for fitness coaches and personal trainers to streamline client nutrition management.

## 🚀 Features
- **Instant Macro Tracking:** Dynamic calculation of calories, proteins, carbs, and fats.
- **Client Customization:** 1-click updates to transition clients between custom meal setups.
- **SaaS-Ready Design:** Modern UI/UX with smooth interactions tailored for business-to-client branding.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom Dark Theme & Layouts)
- JavaScript
